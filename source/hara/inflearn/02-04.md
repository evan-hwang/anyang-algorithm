# 인프런 02-04문제
- 피보나치 수열
  - URL : https://cote.inflearn.com/contest/10/problem/02-04

### 문제 이해하기

**Input**

	* 숫자 N ( 3<=N<=45)

**Output**

* 피보나치 수열 N개

### 문제 접근 방법

* A + B = C 
  * A= B , B =C
    * A+ B = C ....(반복)

### 구현 배경 지식

	* 피보나치 수열 : 앞의 2개의 수를 합하여 다음 숫자가 되는 수열
	ex) 1 1 2 3 5 8 13

### 문제를 해결한 코드

**Code1**

​	*  1,1을 초기값으로 두고 , 앞항과 뒤항을 더해서 수열을 만든다 (배열)

```java
import java.util.*;

public class main2 {
	public int[] solution(int n){
		int[] answer = new int[n];
		answer[0]=1;
		answer[1]=1;
		for(int i=2; i<n; i++) {
			answer[i]=answer[i-2]+answer[i-1];
		}
				
		return answer;
	}
	
    public static void main(String[] args){
    	main2 T = new main2();
    	Scanner kb = new Scanner(System.in);
    	int n = kb.nextInt();
    	for(int x : T.solution(n)){
    		System.out.print(x+" ");
    	}
    }
}
```

```java

import java.util.*;

public class main2 {
	public void solution(int n){
		//배열을 사용하지 못할 때
		int a=1, b=1, c;
		System.out.print(a+" "+b+" ");
		for(int i=2; i<n; i ++) {
			c= a+b;
			System.out.print(c+" ");
			a=b;
			b=c;
		}
	}
	
    public static void main(String[] args){
    	main2 T = new main2();
    	Scanner kb = new Scanner(System.in);
    	int n = kb.nextInt();
    	T.solution(n);
    }
}

```
