### 문제 이해하기
사용자로 부터 입력 받은 단어를 뒤집어서 출력하라.
### 문제 접근 방법
문자열(String)을 문자배열(char[])로 변환 후 서로 대칭대는 index를 swap하여 값 변경 후 문자열(String)로 변환
### 구현 배경 지식
- swap에 대한 이해
- StringBuilder와 String의 차이점
- 문자열을 char[] '문자 배열'로 변환 시키는 방법
### 문제를 해결한 코드
```java
import java.util.ArrayList;
import java.util.Scanner;

public class ReverseString {
    public static void main(String[] args) {
      // String[] str = {"good", "Time", "Big", "study"};
      Scanner scan = new Scanner(System.in);
      int n = scan.nextInt();
      String[] str = new String[n];
      for(int i=0; i<n; i++){
        str[i] = scan.next();
      }
      System.out.println("===============================");
      for(String x : solution(str)){
        System.out.println(x);
      }
      scan.close();
    }
    public static ArrayList<String> solution(String[] str){
      ArrayList<String> answer = new ArrayList<>();

      // 내장함수 사용 - 강의 풀이
      for(String x : str){
        String tmp = new StringBuilder(x).reverse().toString();
        answer.add(tmp);
      }

      // 코드구현 단어 뒤집기 1 - 강의 풀이
      for(String x : str){
        char[] s = x.toCharArray();
        int lt = 0;
        int rt = x.length() - 1;
        while(lt < rt){
          char tmp = s[lt];
          s[lt] = s[rt];
          s[rt] = tmp;
          lt++; 
          rt--;
        }
        String tmp = String.valueOf(s);
        answer.add(tmp);
      }

      // 코드구현 단어 뒤집기 2 - 본인 구현
      for(String x : str){
        char[] ch = x.toCharArray();
        String tmpStr = "";
        for(int i=ch.length-1; i>=0; i--){
          tmpStr += ch[i];               
        }
        answer.add(tmpStr);
      }
      return answer;
    }
}
```
