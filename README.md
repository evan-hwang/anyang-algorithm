# 🛠 안양인들의 알고리즘 스터디
## 1. 디렉터리 구조
```
├── README.md             // 저장소 리드미
├── source                // 과제 코드 저장소
│   ├── hyuckjin          // 과제 제출자
│   │   ├── programmers   // 문제 사이트
│   │   ├── backjun       
│   │   └── inflearn      
│   │       ├── 01-02.md  // 문제 풀이 (파일명은 문제번호 권장, 템플릿 기반 작성)
│   │       └── ...
│   └── ...
├── study_how.md          // 알고리즘 공부 방법
├── template.md           // 문제 풀이 정리를 위한 템플릿
└── template_example.md   // 템플릿 예시
```

## 2. 문제 풀이 템플릿
### 제목
```
[상태색상값] 문제명.md
```
- 상태 색상값
  - 한번에 푼 경우 : 🟢
  - 한번에 못 푼 경우 : 🟡
  - 이해가 안가는 경우 : 🔴
- 문제명 
  - 최대한 문제 번호로 입력
  - 없는 경우 문제 제목
